# Install Redis
- `brew install redis`
- `brew services start redis`

# Redis CLI
- `redis-cli`
- `KEYS *` return all keys
- `TYPE key_name_here` get key type
- `HGET key_name_here field` get key

# Run `RedisExampleApplication.java`
